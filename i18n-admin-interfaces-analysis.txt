## 3. Components
Most Admin-specific components are currently implemented inline within the pages listed above (e.g. StatCard, RoleBadge, UserPanel, CreateAdminModal, etc.).
No external shared Admin-only components were found in `frontend/src/components/admin` as the directory does not exist.
Shared components from `frontend/src/components/ui` and `frontend/src/components/common` are already covered in the previous i18n scope.

## i18n Strategy Confirmation
- **Loading:** Translations are loaded via `react-i18next` using the `translation` namespace from JSON files.
- **Namespace:** Single namespace "translation" used platform-wide.
- **Key Pattern:** Domain-based nesting (e.g. `admin.users.governance.title`).
- **Implementation:** Use `useTranslation` hook in client components and `t()` function for keys.
